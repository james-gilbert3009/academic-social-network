# Academic Social Network (Bachelor Thesis Project)

## 📌 Overview
This project is a web-based academic social networking platform designed for students and faculty members. It allows users to create profiles, share academic interests, and interact within a university environment.

## 🎯 Features Implemented
- User Authentication (Register/Login)
- Login with Email or Username
- Profile Setup Page
- Profile Page with Edit Functionality
- Profile Picture Upload
- Profile Completion Reminder
- JWT-based Authentication

## 🛠️ Tech Stack
### Frontend
- React.js (Vite)
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Multer (file upload)
